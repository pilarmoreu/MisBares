· La aplicación MIS BARES ha sido creada por Pilar Moreu Falcón.

· Debe ejecutarse con el software de desarrollo Android Studio. 

· En el proyecto no es necesario importar las librerías del directorio MisBares/app/libs
puesto que no hay (únicamente se emplea la librería por defecto 'appcompat-v7:23.3.0'.

· Las imágenes utilizadas en la aplicación se encuentran en el directorio MisBares/app/
src/main/res en las carpetas drawable y mipmap.

· Para poder importar/exportar los datos de la base de datos es necesario permitir el 
almacenamiento desde el teléfono (Ajustes > Aplicaciones > Mis Bares > Permisos). Sin
embargo, al usuario se le informa de este dato desde la aplicación (mediante toast).
